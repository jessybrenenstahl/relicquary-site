# relicquary-site

Marketing + company site for **Relicquary**, a local-first memory OS for AI agents, by RQ MCP LLC. Served by GitHub Pages at [relicquary.com](https://relicquary.com).

## Layout

- `index.html`: home / product page (conversion at `#get`, apps at `#apps`)
- `tools.html`: the 57-tool MCP catalog (machine twin: `tools.json`)
- `record.html`: the ledger. Dated releases, recorded runs, hardware verification, and the register of what has not run. Every evidence chip on the site resolves here.
- `stack.html`: company front door + AIM (mirrored later to rqmcp.llc / aim.relicquary.com; its cross-links are absolute for that reason)
- `touchlink.html`, `support.html`, `privacy.html`, `terms.html`
- `assets/styles.css`, `assets/site.js`: all styles and shared behavior
- `CNAME`: custom domain (`relicquary.com`); `.nojekyll`: serve files as-is

## Agent layer (generated, do not hand-edit)

`<page>.md` twins, `relicquary-corpus.md`, `relicquary-md.zip`, `llms-full.txt`, `relicquary.md`, and the `data-stamp` spans are generated by:

```bash
python3 scripts/build-agent-layer.py   # requires pandoc
```

Run it after any HTML content change. CI (`.github/workflows/agent-layer.yml`) regenerates on push and fails on drift. Hand-maintained agent files: `llms.txt`, `scripts/llms-orientation.md` (the first half of `llms-full.txt`), `tools.json` (regenerated from the Relicquary binary's `mcp-schema` output when the tool surface changes).

## Local preview

```bash
python3 -m http.server 8080
# open http://localhost:8080
```

## Deploy

Push to `main`; GitHub Pages rebuilds automatically. Confirm with `curl -I https://relicquary.com`.

## Domains

- `relicquary.com`: this repo (DNS at Squarespace; never drop the MX/DKIM/SPF email rows)
- `rqmcp.llc`: serves the `rqmcp-llc` repo directly (no redirect), DNS at Namecheap
- `aim.relicquary.com` / `touchlink.relicquary.com`: planned mirrors, DNS pending
- Contact alias used across the site: `hello@relicquary.com` (routes in Google Workspace)

## Copy rules (owner-set)

No em dashes. No manipulation or hype; describe what the thing is and does. Never claim the system learns, grows, or self-improves. Availability stated once, in its home (`index.html#apps`, `record.html`). Every run-derived claim carries a `.receipt-cite` chip that resolves to a dated `record.html` entry.
