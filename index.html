What it is

A durable memory layer that lives on your disk.

Relicquary is a single, local binary that stores typed memory, a knowledge graph, retrieval, synthesis, and receipts as plain SQLite and Markdown files. Agents reach it over the Model Context Protocol; humans use the CLI. Nothing is hosted, nothing is hidden, and every mutation leaves an auditable trail you can read by hand.

It's built for work that spans many sessions: backlogs route, sessions open and close with durable receipts, and context packs & handoffs are generated from local state — so an agent can pick up exactly where it left off.

# capture a memory relicquary remember mystore "API keys live in 1Password, vault: Eng" # recall it later — hybrid, explainable relicquary recall mystore "where are the api keys" → [fact · current · 0.94] API keys live in 1Password… # or expose it to any agent over MCP relicquary mcp-server
